Course Management System

Project Description

This project is a simple Course Management System developed using Python and SQL.
It helps to manage student details, courses, department, and college information.
The system allows users to store, view, and manage student records using a database.

Technologies Used

- Python
- SQL Server
- PyCharm
- GitHub

Features

- Add new student details
- View student records
- Manage course information
- Store department and college details
- Database integration using SQL

Database

The project uses a SQL database named StudentDB which contains a table called Students.

Table Fields:

- StudentID
- Name
- Age
- Department
- College
- Course

How to Run the Project

1. Open the project in PyCharm.
2. Run the Python file.
3. Ensure the SQL Server database is connected.
4. Execute the program to manage student data.
